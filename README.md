# Movies, a Flutter APP

Flutter app for searching info about movies such as raitings, overviews, casting/actors and more.

Flutter components / widgets that i used: 

- CardSwipers
- API consumption
- Provider
- Futures 
- Streams 
- Stream Builders
- Hero Animation
- Search Delegate
- Bloc Pattern
